O Sistema de Gerenciamento de Condomínios Conflow possui diversas funcionalidades. Esse guia tem a funcionalidade de dar dicas básicas de como utilizar cada uma delas.

1. Tela principal

A tela principal da acesso a todas as outras telas. Ela possui os botões "Avisos", "Conectar", "Pesquisar", "Cadastrar", "Configurar", "Ajuda" e "Sair".

 

2. Avisos

Os avisos são configurados pelo administrador. Ao conectar-se a uma conta, os avisos serão a primeira coisa a ser exibida na tela.

 

3. Conectar

Conecta-se a uma conta usando um nome de usuário e senha. É necessário conectar-se para poder usar o Conflow.

 

4. Pesquisar

Procura por um Proprietário, Prédio, Bloco, etc e os filtra usando critérios escolhidos pelo usuário. É possível editar os proprietários clicando no botão "Editar", que fica ao lado de cada um dos resultados.

 

5. Cadastrar

Cadastra um Proprietário ao informar os dados requisitados. É obrigatório preencher todos os campos.

 

6. Configurar

Muda configurações preferenciais do usuário como tamanho da fonte e estilo visual do programa.

 

7. Ajuda

Exibe esta tela.

 

8. Encerrar

Encerra o programa e retorna para a área de trabalho.

 

9. Teclas de atalho

Para selecionar rapidamente os menus, é possível usar as seguintes teclas de atalho:

Tecla   Função
F1     Avisos
F2   Conectar
F3   Pesquisar Proprietário
F4   Cadastrar Proprietário
F5   Configurar
F6   Ajuda
F12   Sair
